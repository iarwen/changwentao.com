Service层测试的基类

注解介绍：
@RunWith：单元测试的执行类，主要提供Spring运行时的环境
@SpringBootTest:执行SpringBoot的入口类
@Transactional：配置事务，在单元测试用控制事务
@Rollback：回滚方式执行单元测试
